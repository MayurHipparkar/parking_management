# Parking Management System (MVC)

## Overview

The Parking Management System is a web-based application developed using Java, Spring MVC, JSP, Maven, and MySQL. The system helps manage vehicle parking operations efficiently by providing role-based access for Admins and Guards.

## Features

### Authentication & Session Management

* Secure Login System
* Session-based Authentication
* Role-Based Access Control (Admin and Guard)

### Admin Module

* Add, Update, Delete Guards
* Manage Parking Slots
* View Parking Details
* Manage Payment Information
* Monitor Vehicle Parking Activities
* Generate Reports

### Guard Module

* Add Vehicle Entry
* Allocate Parking Slots
* Manage Vehicle Exit
* Calculate Parking Charges
* Generate Parking Reports
* View Assigned Parking Information

### Vehicle Management

* Vehicle Registration
* Vehicle Entry Tracking
* Vehicle Exit Tracking
* Parking Slot Assignment
* Parking History Maintenance

### Dashboard

After successful login, users are redirected to the dashboard where they can access functionalities based on their role and permissions.
